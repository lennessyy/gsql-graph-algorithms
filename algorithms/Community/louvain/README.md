## Installation 

### Replace `<Louvain Algorithm>` with desired algorithm listed above 

#### Via TigerGraph CLI

```bash
$ tg box algos install <Louvain Algorithm>
```

#### Via GSQL terminal

```bash
GSQL > BEGIN
# Paste <Louvain Algorithm> code after BEGIN command
GSQL > END 
GSQL > INSTALL QUERY <Louvain Algorithm>
```
