## Installation 

### Replace `<Harmonic Algorithm>` with desired algorithm listed above 

#### Via TigerGraph CLI

```bash
$ tg box algos install <Harmonic Algorithm>
```

#### Via GSQL terminal

```bash
GSQL > BEGIN
# Paste <Harmonic Algorithm> code after BEGIN command
GSQL > END 
GSQL > INSTALL QUERY <Harmonic Algorithm>
```
